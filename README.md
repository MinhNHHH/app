## Requirement
1. django== 3.2.3
2. djangorestframework
3. django-rest-swagger
4. django-cors-headers
5. django-rest-knox
6. werkzeug
7. axios
8. react-router-dom
## Setup

- Clone Repository
- `cd back-end`
- `pip install -r requirements.txt`
- `python manager runserver`
- `cd front-end`
- `npm install`
- `npm start`

## Features
- Login/Register
- Tracking budget and tracsaction monthly 
- Add incomes/costs to month
